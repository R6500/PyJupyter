=======================================================
 Main Folder Contents
=======================================================

This is the main folder of the PyJupyter project

It contains the following subfolders:

Jupyter : Jupyter notebooks for the tutorial and
          the system documentation

Scripts : Operating System scripts

Setup Docs : Documentation for PyJupyter installation

Jupyter.bat : Windows script to launch PyJupyter

Jupyter.sh : Linux script to launch PyJupyter
